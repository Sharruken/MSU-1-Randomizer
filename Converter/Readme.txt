# Preparation

This converter uses FFMPEG for MP3 and MP4 Files, to make this work, go download FFMPEG at
https://www.ffmpeg.org/
and extract it to this folder

# Usage

1. Put any MP3s and MP4s into a folder names 'MP3s'
2. Put any .wav, .flac and .ogg files into a folder named 'Music'
3. Run Converter.py
